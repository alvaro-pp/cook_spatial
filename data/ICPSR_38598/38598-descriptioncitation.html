Project Description:

Summary: This collection contains measures of land cover (e.g., low-, medium-, or high-density development, forest, wetland, open water) derived from the National Land Cover Database (NLCD) and aggregated by United States census tract and ZIP code tabulation area (ZCTA). For each land type, land cover is measured both in total square meters and as a proportion of all land of that type within the tract or the ZCTA.

Methodology:

Study Purpose: The purpose of these data are to measure land cover based on National Land Cover Database specifications in the continental United States.

Study Design: The research team used measures derived from the 2016 National Land Cover Database (NLCD) spatial data released by the U.S. Geological Survey. The team then aggregated the data into annual measures by census tract and ZIP code tabulation area. The data are based on the NLCD data releases (2001, 2003, 2006, 2008, 2011, 2013, 2016) and have been interpolated between adjacent years. The variable INTP_FLAG in both datasets indicates whether values have been interpolated.

Time Method: Longitudinal

Universe: All census tracts and ZIP code tabulation areas in the continental United States. No data is available for Alaska, Hawaii, or the United States territories.

Unit(s) of Observation: ZIP code tabulation area, Census tract

Data Source: Data and documentation for the ZCTA-level data were originally deposited in openICPSR project 128862., Data and documentation for the census tract-level data were originally deposited in openICPSR project 110663., Census tract boundaries were obtained from the 2010 TIGER/Line shapefiles., Multi-Resolution Land Characteristics Consortium. "National Land Cover Database (CONUS), All Years," 2016. https://www.mrlc.gov/data/nlcd-land-cover-conus-all-years., ZIP code tabulation area boundaries were obtained from the TIGER/Line shapefiles, 2010 census geography, 2019 version.

Data Type(s): aggregate data

Description of Variables: Variables include measures of green space and blue space (e.g. open water, deciduous forest, evergreen forest) and categories of developed space (e.g. low intensity, medium intensity, high intensity development). For each category of land cover, measures include both the total land cover in each census tract/ZIP code tabulation area (ZCTA) in square meters and the proportion of land coverage of that type in each tract/ZCTA.
